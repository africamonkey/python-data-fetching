# 相关内容推荐 / 文章相似度检测

## 余弦相似度

余弦相似度用向量空间中两个向量夹角的余弦值作为衡量两个个体间差异的大小。余弦值越接近1,就表明夹角越接近0，也就是两个向量越相似。

## 文章的余弦相似度

$cos(\theta) = \frac{\sum\limits_{i = 1}^{n} (x_iy_i)}{\sqrt{\sum\limits_{i=1}^{n}x_i^2} \times \sqrt{\sum\limits_{i=1}^{n}y_i^2}}$
